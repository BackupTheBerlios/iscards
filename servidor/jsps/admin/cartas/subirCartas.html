Añadir/Modificar cartas


A través de este asistente podrá subir cartas generadas por el editor de cartas de Génesis a la base de datos, para que los jugadores puedan disponer de ellas.

Para empezar, introduzca la ruta donde se encuentra el archivo .car generado por el editor de cartas y pulse Aceptar para continuar.
